Recursividade em Java - Introdução às Estruturas de Dados

Enunciado
Criar uma aplicação em Java com uma função recursiva que, recebendo um vetor de números inteiros, o tamanho do vetor e o valor da última posição como o primeiro menor valor, retorne o menor valor contido neste vetor.

O código também deve apresentar comentários explicando:
- A condição de parada;
- Como a função chama ela mesma (relação de chamada).

Tecnologias: Java

O que foi praticado:
- Recursão
- Manipulação de vetores
- Comparação de valores
- Organização em pacotes
- Estrutura básica de projetos Java
